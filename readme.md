# VeSkills — Verified Skills

**Слоган:** «Покажи, что ты умеешь, а не рассказывай, чему тебя учили»

Открытая, децентрализованная система верификации навыков с доказательствами и цепочкой поручительства. VeSkills — это не платный сертификат, а реальное подтверждение ваших компетенций пруфами.

---

## Идеология

- **Пруфы вместо сертификатов.** Хочешь подтвердить навык — покажи видео, фото, протокол, деталь. Не можешь доказать — не лезь.
- **Парампара вместо HR.** Навыки подтверждают живые люди, а не платные центры сертификации. Видна вся цепочка поручительства.
- **Меритократия вместо демократии.** Голосуют только равные или старшие по уровню. Лайк от дилетанта не весит ничего.
- **Децентрализация.** Нет единого центра сертификации. Любая Школа может создать ветку навыков и подтверждать учеников.
- **Открытость.** Open-source. Бесплатный вход. Публичные профили. Никаких платных подписок.
- **Универсальность.** Навыки описываются через измеримые результаты, а не через привязку к конкретному оборудованию или стране.

---

## Видение

Заменить рынок резюме и платных сертификатов на экосистему открытых верифицированных навыков. Создать стандарт, который переживёт любое государство, любой HR-отдел, любой HeadHunter.

---

## Стек

- **Бэкенд:** Node.js + Express + pg (чистый SQL, параметризованные запросы)
- **Фронтенд:** Vue 3 (Composition API) + PrimeVue
- **База данных:** PostgreSQL
- **Развёртывание:** Docker (on-premise, без облаков)

## Ключевые принципы

- **Вассал моего вассала — не мой вассал.** Руководитель управляет только прямыми подчинёнными.
- **Два срока действия навыка.** Глобальный (от skills) и локальный (от руководителя). Действуют оба.
- **Пруфы обязательны.** Без доказательств уровень навыка не является публично подтверждённым.
- **Верификация только равными и старшими.** Оценивать может только тот, кто сам владеет этим навыком на том же или более высоком уровне.
- **Уровень не может быть снят руководителем.** Только по сроку годности или через арбитраж.

## Роли

| Роль | Права | Ограничения |
|---|---|---|
| **superadmin** | Полный доступ | Не оценивает навыки |
| **admin** | Управление пользователями и оргструктурой | Не назначает других админов |
| **trainer** | Управление онтологией, допусками и обучениями | Не оценивает скиллы |
| **manager** | Управление компетенциями подчинённых | Только прямые подчинённые |
| **employee** | Загрузка пруфов, просмотр и оценка | Только своих или равных |
