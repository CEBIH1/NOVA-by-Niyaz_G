# Roadmap (План развития VeSkills)

## v0.1.0 — MVP (Минимально жизнеспособный продукт)

**Цель:** Показать работающую матрицу компетенций на одном заводе.

- [x] Базовая схема БД (миграция 001)
- [x] Базовые API-маршруты
- [x] Миграция 002 (пруфы, верификации, лайки)
- [x] Макет Dashboard, иерархия, навигация
- [x] Карточка сотрудника (радар, таблица навыков, детали)
- [x] Карточка подразделения (радар, матрица компетенций)
- [ ] Регистрация и аутентификация (JWT)
- [ ] CRUD для сотрудников и навыков
- [ ] Подключение фронтенда к API
- [ ] Загрузка пруфов
- [ ] Бинарная верификация
- [ ] Уведомления о сроках

## v0.2.0 — Доказательная база

**Цель:** Превратить матрицу в инструмент с реальными пруфами.

- [ ] Лайки/дизлайки пруфов
- [ ] Просмотр пруфов по навыку (кросс-цеховой)
- [ ] Аудит-лог (журнал всех изменений)

## v0.3.0 — Чек-листы и качество

**Цель:** Внедрить расширенную верификацию.

- [ ] Шаблоны чек-листов
- [ ] Обязательные отметки при верификации
- [ ] Автоматическая эскалация повторных замечаний

## v1.0.0 — Публичный релиз

**Цель:** Открыть проект для мира.

- [ ] Публичная страница профиля сотрудника
- [ ] Документация для пользователей
- [ ] Презентация для конференции
- [ ] Деплой (Docker)
- [ ] Публикация на GitHub
